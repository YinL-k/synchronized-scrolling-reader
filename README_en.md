# VSCode‑style Side‑by‑Side Viewer

This repository contains a self‑contained HTML file that implements a VSCode‑style, two‑pane code viewer with synchronized scrolling. It lets you load two text or code files side‑by‑side so that each pane scrolls in sync with the other, making it easy to compare documents or translations. The viewer supports both dark and light themes, customizable font sizes, basic syntax highlighting and line numbers.

## Features

- **Synchronized scrolling:** Scroll either pane and the other follows along.
- **Two‑panel layout:** Load two different files at once for comparison.
- **Theme toggle:** Switch between dark and light themes to suit your preference.
- **Font adjustment:** Increase or decrease font size using the toolbar buttons.
- **Syntax highlighting:** Basic highlighting for common programming languages.
- **Save and load:** You can load local files via drag‑and‑drop, and save your changes.

## Getting started

1. Clone or download this repository.
2. Open `Synchronized scrolling reader(1).html` in your browser.
3. Use the toolbar to load your files by dragging them into each pane or using the file picker.
4. Use the buttons on the toolbar to adjust font size, toggle themes or save your work.

No extra dependencies are required; everything runs directly in the browser.
