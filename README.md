# VSCode风格双栏查看器

本仓库包含一个自包含的HTML文件，实现了VSCode风格的双栏代码查看器，可实现同步滚动。它允许你同时加载两个文本或代码文件，实现双栏对比，两栏的滚动保持同步，方便比较不同文档或译文。

## 功能特点

- **同步滚动：** 滚动任意一栏，另一栏跟随。
- **双栏布局：** 同时加载两个不同文件进行对比。
- **主题切换：** 支持暗色和浅色主题。
- **字体调节：** 工具栏按钮可增大或减小字体大小。
- **语法高亮：** 提供常规编程语言的基础语法高亮。
- **保存与加载：** 可通过拖拽或文件选择加载本地文件，并支持保存修改。

## 快速开始

1. 克隆或下载此仓库。
2. 在浏览器中打开 `Synchronized scrolling reader(1).html`。
3. 使用工具栏将文件拖拽到左右面板或使用文件选择器加载。
4. 使用工具栏按钮调整字体、切换主题或保存修改。

本项目无需额外依赖，所有功能均在浏览器中运行。
